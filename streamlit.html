Project Overview

In this project,technical analysis is used to examinate the stock market including forex and commodities market.

This application allows the users to choose indicators to indicate the buy and sell signals
of any tickers and predict the next day close price .
This project is divided into three parts:
  • Data Collection
    • To start up the application,stock market data is loaded from Yahoo Finance with pandas datareader.
    In order to make the analysis be more interactive,users can input their interested ticker,
    period and also indicators.
    Then the datareader function will take the parameters and retrieve the related data from Yahoo Finance.

  • Data Analysis and Visualisation
    • In this part,the close price of the ticker will be analysed byBollinger Bands with RSI,Moving Average
    Convergence Divergence and On-balance Volume    . Before implementing the function of calculating the indicators,Simple Moving Average,Exponential Moving Avergae,
    Standard Deviation and Profit function     were created which would be generally used in the indicators.
    In these functions,window function,'ewm' and 'rolling', are frequently used.
    To visualise the results,plotly is used to demonstrate the indicators.For the Buy and sell signals,
    scatters and table with mapping style are used to displayed on the indicators.

  • Prediction with Regression
    • In this part next day close price is predicted by Support Vector Regression from sklearn with Radial Basis
    function kernel, Polynomial regressions and Linear regression    .
