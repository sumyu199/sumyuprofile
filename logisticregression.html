Project Overview

In this project,the 2012 Olympics Althletes data is investigated to examinate the correlations of physical features and possibility of winning a competition.
For this project,I am using different plots and hypothesis tests to evaluate the correlations
and using logistic regression to build a predictor model for the classification of recieved a reward or not.

This project is divided into three steps:
  • Clean the Data and Exploratory Analysis

    • For a regression model,the features are ideally to be continuous variables hat have an affect
    on our dependent variable or binary.
    Therefore, in this part,I filled all NA values with mean values and transform Medal column into binary
    structure.Also,for a better understanding of the relationship between Medals and features,I visualised
    the features of both medalists and non-medalists with QQ plots, Histograms and Boxplot and compared them.
    Futhermore, I used t-test to see whether the medalists and non-medalists are having different mean features and
    all the results are positive so we can prove that medalists should have beneficial features to win.

  • Split the Data into Training and Testing Sets

    • In this part,to start with,I set.seed(123) to ensure the partition is reproducible and the ratio of training and testing sets is 8:2.
    For the model,the inital features are Height,Weight and Age,the label is Medal with numerical structure.

  • Create and and Evaluate the Models

    • In this part,I decided to use logistic regression glm to build the model because logistic regresssion is suitable for classification when the the dependent variable is binary and we only have few features.
    To decide which features should be used,z-test and p-value are used to indicate the significance of
    association between label and features.
    Eventually,Height is having the most significant association with the label.As a results,Height is the only features
    of the model and the classification prediction accuracy is about 81%.
    However,there are many different variables which are also related to be rewarded or not for example length of training
    years,experiences and talents so for the future improvement,I might conclude more features in the model.
