Project Overview

In this project,the historical data of titanic passengers is trained to create a predictor to predict
whether the passenger survived or not.
For this project,I am using scikit-learn to build different predictors and model.
The final predictor decided to be Decision tree which is a kind of supervised learning to process
the classification.

This project is divided into three steps:
  • Clean the Data and Exploratory Analysis

    • For a tree decision model,the features are ideally to be continuous variables hat have an affect
    on our dependent variable or binary.
    Therefore, in this part,I transformed sex,cabin and title into binary using lamba
    and fill the nan of the data with numpy.
    Also,for a better understanding of the relationship between survival and features,I exam the
    correlation of the features and visualised the probability of survival of different features.

  • Data Selection and Split the Data into Training and Testing Sets

    • In this part,I selected different sets of features to create several models and split the
    data into training and testing sets with random state = 1 to ensure the sets are reproducible
    which is beneficial to compare the results of different models.

  • Create and and Evaluate the Models

    • In this part,I created both random forest classifier and decision tree classifier.
    To decide which classifier should be used,score function is used to indicate the efficiency
    of forest classifier and the decision tree classifier with different depth and different features.
    Eventually,decision tree with the best depth preforms better than forest.
